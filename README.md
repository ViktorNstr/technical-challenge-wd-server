# Project 3 | Technical Challenge WD - Backend

## Description

This project focuses on building a Web application (Backend). 
We are using ExpressJS. 
A REST API backend with routes is created.
There is 1 model, Phone. 


## Github Repo Technical Challenge WD Client

[Technical Challenge WD Client: ](https://github.com/technical-challenge-wd-viktornstr/technical-challenge-wd-client)



## Instructions

This project was installed using the command:

```bash
$ npm install express
```

## Run this app

In order to run this app please install all the dependencies:

```bash
$ npm install cors
```

And run the app with 

```bash
$ node app.js
```

Now you can start developing.
